call

fun call(users: List<String>, callOptions: CallOptions?, onCallCreationObserver: OnCallCreationObserver): Unit

Creates a call with the users and the options provided as arguments. This is an asynchronous process and success or failure will be reported in the observer provided as argument.

Parameters

users - List of users to call, identified by the userAlias

callOptions - Optional call options

onCallCreationObserver - The observer to be notified when the call has been created or if it has failed
