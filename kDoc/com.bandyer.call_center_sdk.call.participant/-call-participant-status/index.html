CallParticipantStatus

enum class CallParticipantStatus

Class which represents the status of the user in the call

Enum Values

ANSWERED

The user has answered the call

DECLINED

The user has declined the call

INVITED

The user has been invited

ERROR

The user has encountered an error

DO_NOT_DISTURB

The user declined the call because does not want to be disturbed

NO_ANSWER

The user did not interact with the call he received ( Ex: a timeout has occurred )

TIMED_OUT

The has not joined the call in time

LEFT

The user has left the call
