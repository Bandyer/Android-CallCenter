Call

interface Call

It is the item representing a call in Bandyer platform.

It manages the lifecycle of a call, from the connection with other participants, till its conclusion.

The object provides information about the current state of the call (e.g if it's dialing, connecting, ended and so on), information about the kind of call it's being performed (e.g. a group call, or a one to one call), and the call direction.

It also provides information regarding call participants and their state within the call (e.g if a participant has answered, declined and so on).

The following operations can be performed on a call object:

  • HangUp: it will end an ongoing call. An optional reason can be specified when hanging up a call.

If is an incoming call then it can also:

  • Answer: it will answer an incoming call, notifying the other participants of the intent of taking part in the conversation.
  • Decline: it will decline an incoming call, notifying the other participants of the intent of not taking part in the conversation. An optional reason can be specified when declining a call.
